## Prostore - Next Js App

## Tech Stack

1. Nextjs
2. Lucid for icons
3. Shadcn for css components
4. Tailwind for styling
5. PostgreSQL - Neon
6. Prima for ORM
7. Zod for Data Validation

### Prima Setup

```js
npm install -D prisma @prisma/client
npx prisma init
npx prisma migrate dev --name init
npx prisma generate
npx prisma studio // open prisma
```
